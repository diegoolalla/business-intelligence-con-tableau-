# Screenshots de Dashboards

Esta carpeta contendrá las capturas de pantalla de los dashboards creados en Tableau.

## Archivos Esperados

### Screenshots de Dashboards
- `dashboard_overview.png` - Dashboard principal con KPIs
- `dashboard_riesgo.png` - Dashboard de análisis de riesgo
- `dashboard_financiero.png` - Dashboard de análisis financiero
- `mapa_geografico.png` - Vista detallada del mapa
- `evolucion_acumulada.png` - Gráfico de evolución temporal

### Screenshots de Tableau Cloud (Bonus)
- `tableau_cloud_dashboard.png` - Captura del dashboard publicado en Cloud
- `tableau_cloud_url.txt` - URL pública del dashboard (si aplica)

## Instrucciones para Capturas

### En Tableau Desktop
1. Abrir el dashboard deseado
2. Dashboard → Export Image
3. Formato: PNG
4. Resolución: 1920x1080 o mayor
5. Guardar en esta carpeta

### En Tableau Cloud
1. Publicar el workbook
2. Abrir en navegador
3. Captura de pantalla completa
4. Guardar como `tableau_cloud_dashboard.png`
5. Copiar URL y guardar en `tableau_cloud_url.txt`

## Notas
- Usar nombres descriptivos para las imágenes
- Mantener alta resolución para claridad
- Incluir en el .twbx final si es necesario
